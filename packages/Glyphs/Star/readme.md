# Install

> npm install @paprika/glyphs
> or
> yarn add @paprika/glyphs

# How to use it

```js
import React from 'react';
import Star from '@paprika/Glyphs/Star';

then use <Star {...props} />
```

## Api

```js
<Star
  ariaText="..."
  borderColor="#e5e871"
  className="..."
  fillColor="#faff7a"
  size={30}
/>
```
