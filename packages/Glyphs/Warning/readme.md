# Install

> npm install @paprika/glyphs
> or
> yarn add @paprika/glyphs

# How to use it

```js
import React from 'react';
import Star from '@paprika/Glyphs/Warning';

then use <Warning {...props} />
```

## Api

```js
<Warning
  ariaText="..."
  borderColor="#e0841a"
  className="..."
  fillColor="#f4a142"
  size={30}
/>
```
